EduTrack

Application Sitemap

Version 1.0 | April 2026

Overview

EduTrack has four distinct interface zones based on user role: Public (unauthenticated), Admin, Teacher, and Kiosk. Each zone has its own set of routes and UI. Navigation is handled by React Router with protected routes guarding each zone.

Site Structure Tree

/ (App Root)

→ /login — Login Page

→ /admin — Admin Dashboard (home)

→ /admin/students — Student List

→ /admin/classes — Class List

→ /admin/classes/new — Create Class

→ /admin/classes/:id — Class Detail & Edit

→ /admin/teachers — Teacher List

→ /admin/teachers/new — Add Teacher

→ /admin/teachers/:id — Teacher Profile & Edit

→ /teacher/classes — My Classes

1. Public Routes

Route / Path

Page Name

Description & Key UI Elements

/login

Login Page

Email + password sign-in form. Redirects to /admin, /teacher, or /kiosk based on role. Includes 'Forgot Password' link.

2. Admin Zone (/admin/*)

Accessible only to users with the Admin role. Protected route wrapper redirects unauthorized users to /login.

Route / Path

Page Name

Description & Key UI Elements

/admin

Admin Dashboard

Overview cards: total students, classes today, active sessions, absence alerts. Quick links to key actions.

/admin/students

Student List

Searchable/filterable table of all students. Columns: name, grade, class, fingerprint status, active. Actions: Add, View, Deactivate.

/admin/students/new

Add Student

Form: name, grade, class assignment, guardian email. On save, redirects to fingerprint enrollment.

/admin/students/:id

Student Profile

Full student detail: info, class history, attendance summary chart, absence count. Edit student info.

/admin/students/:id/enroll

Fingerprint Enrollment

Kiosk-style scan interface for enrolling a student's fingerprint. Shows step-by-step: scan → confirm → save.

/admin/classes

Class List

Table of all classes: name, teacher, schedule, room, student count. Actions: Create, Edit, Delete.

/admin/classes/new

Create Class

Form: class name, subject, teacher assignment, schedule (days/times), room.

/admin/classes/:id

Class Detail

Class info, enrolled students list, session history, attendance rate chart.

/admin/teachers

Teacher List

Table of all teachers: name, email, assigned classes. Actions: Add, Edit.

/admin/teachers/new

Add Teacher

Form: name, email, password setup, class assignments.

/admin/teachers/:id

Teacher Profile

Teacher info and class assignments. Edit capability.

/admin/reports

Reports & Analytics

Filters: by class, date range, student. Outputs: attendance rate table, bar charts, absence list. Export to CSV/PDF.

/admin/settings

System Settings

Configure: absence threshold %, notification emails, session time windows, kiosk PIN.

3. Teacher Zone (/teacher/*)

Accessible only to Teacher role. Teachers only see data for their assigned classes.

Route / Path

Page Name

Description & Key UI Elements

/teacher

Teacher Dashboard

Today's classes with session status (upcoming / active / done). Absence alerts for their classes. Quick 'Open Session' button.

/teacher/classes

My Classes

List of the teacher's assigned classes with schedule and student count.

/teacher/classes/:id

Class Detail

Class info, enrolled students, past session list, attendance rate.

/teacher/classes/:id/session

Active Session View

Real-time attendance board. Shows each student as Present (green) / Absent (grey) as they scan in. Manual override button. Close Session button. Live updates via AppWrite Realtime.

/teacher/reports

My Reports

List of classes with link to attendance report for each.

/teacher/reports/:classId

Class Attendance Report

Per-class attendance grid: rows = students, columns = sessions. Color-coded cells. Export button.

4. Kiosk Zone (/kiosk/*)

Full-screen lock-screen UI intended for dedicated classroom tablets or PCs with a fingerprint scanner attached. Minimal chrome — no navigation bar. The kiosk device is logged in with a dedicated kiosk account.

Route / Path

Page Name

Description & Key UI Elements

/kiosk

Kiosk Home

Shows list of currently active sessions. Staff selects the correct class to activate the scan screen. PIN protected to prevent students from changing class.

/kiosk/:sessionId

Active Scan Screen

Large prominent fingerprint icon. Instruction: 'Place your finger on the scanner'. Waiting state with animated pulse. Auto-resets after each scan.

/kiosk/success

Check-In Success

Full-screen green confirmation: student name, photo (optional), timestamp. Auto-dismisses after 3 seconds.

/kiosk/error

Scan Failed

Red error screen with retry instruction. After 3 failures, shows 'See your teacher' message.

5. Navigation Structure

Zone

Navigation Style

Admin

Persistent left sidebar with icons + labels. Sections: Dashboard, Students, Classes, Teachers, Reports, Settings.

Teacher

Persistent top navigation bar. Links: Dashboard, My Classes, Reports.

Kiosk

No navigation. Full-screen single-purpose interface. PIN-protected exit button to return to Kiosk Home.

Public

Minimal: just the login form. No sidebar or nav bar.
